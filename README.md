## Objetivo
Implementar una solución completa de almacenamiento y consultas en MongoDB utilizando el dataset oficial de producción agrícola de Colombia (datos.gov.co).

## Dataset utilizado
- Fuente: https://www.datos.gov.co/Agricultura-y-Desarrollo-Rural/Consolidado-agr-cola-por-municipios-cultivos-anual/van8-pi8b
- Registros importados: 63

## Estructura del proyecto
- `scripts/`: consultas

## Consultas destacadas
- Consultas básicas (inserción, selección, actualización y eliminación de documentos).
- Consultas con filtros y operadores.
- Consultas de agregación para calcular estadísticas (contar, sumar, promediar, etc.).
